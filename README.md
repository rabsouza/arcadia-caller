## Como usar

Fazer o clone do reposítório git e baixar as dependências
```sh
$ git clone [git-repo-url] arcadia-caller
$ cd arcadia-caller
$ mvn clean install
```
Executar o GAE local
```sh
$ mvn appengine:devserver
```
Fazer upload da app para GAE
```sh
$ mvn appengine:update
```
